깃허브에서 remote repository 만들고
로컬에서 폴더 만들고 
cmd 그 로컬 폴더 위치로 이동하고 git clone 주소 입력하면 연동됨 ( 로컬 폴더 안에 연동된 repository가 생성된다 )
git clone은 다운로드 느낌
cmd 로컬 폴더 안에 연동된 repo 위치로 이동하고
파일 생성하거나 파일 내용 변경하면 git add 해야 commit 가능,     여러개는 git add .
git commit -m  "설명하고싶은 내용"
git commit --amend  ->  commit 내용수정
git status ->  변경사항확인 
git push - > remote repository에 올리기
